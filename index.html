GKA - Plataforma Farmacêutica

Este sistema web foi criado para o meu projeto de conclusão de curso. Consiste em um sistema de busca de medicamentos onde o usuário pode comparar preços entre as farmácias mais próximas de sua localização, usando geolocalização. O projeto visa facilitar a busca de medicamentos pelo usuário (além dos recursos pagos) e também gerar maior alcance para as farmácias (que pagam para usar a plataforma). O sistema possui logins como usuário, farmácia e administrador.
